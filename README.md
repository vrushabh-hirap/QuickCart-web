# 🛒 QuickCart - Fast. Simple. Smart Shopping.

QuickCart is a modern Android e-commerce application designed to revolutionize mobile shopping. This repository contains the source code for the QuickCart landing page, a sleek and responsive web portal that showcases the app's features and provides a direct download for the APK.

---

## 🌟 Key Features

- **Secure Authentication**: Safe and reliable user login and registration system with Firebase integration.
- **Smart Product Browsing**: Intuitive product discovery with category filtering and search functionality.
- **Add to Cart System**: Seamless cart management with real-time updates and quantity controls.
- **Smooth Checkout**: Streamlined checkout process with order summaries and payment details.
- **Modern UI Design**: Clean, intuitive interface with smooth animations and responsive layouts.
- **Order Management**: Track orders and view history in one place.

---

## 🛠️ Tech Stack

### 📱 Android Application
- **Language**: Java & XML
- **IDE**: Android Studio
- **Backend**: Firebase (Authentication, Realtime Database)

### 💻 Landing Page
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started (Landing Page)

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vrushabh-hirap/quickcart-website.git
   cd quickcart-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
